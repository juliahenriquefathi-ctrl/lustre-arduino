Descrição do Projeto

O projeto foi desenvolvido para a atividade de Ferramentas Colaborativas, com o objetivo de criar um lustre funcional utilizando LEDs e Arduino. O sistema simula três tipos de iluminação: iluminação contínua, iluminação intermitente automática e iluminação controlada por interação humana.

O projeto aborda fundamentos de eletrônica básica, programação para microcontroladores e trabalho colaborativo usando GitHub e Trello.


Hardware (Componentes Utilizados)

1x Arduino Uno R3

1x Protoboard

3x LEDs

3x Resistores (220Ω ou 330Ω)

Jumpers (macho-macho)

1x Botão (push button)

1x Fonte USB ou alimentação adequada

Software e Dependências

Linguagem

C++ (Arduino)

1. LED 1 – Sempre aceso
Este LED permanece ligado durante toda a execução do código, simulando uma lâmpada fixa.


2. LED 2 – Pisca a cada intervalo definido
O LED pisca automaticamente em intervalos de tempo predeterminados (1s, 3s ou 5s).


3. LED 3 – Pisca ou apaga com interação do usuário
O LED responde ao pressionamento de um botão, alternando entre ligado e desligado.

Instalação e Montagem

Montagem

Conectar cada LED a um pino digital do Arduino com resistores.

Conectar o botão ao pino digital configurado com resistor pull-down ou pull-up.
